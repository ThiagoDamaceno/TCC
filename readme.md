## Dependências para executar o projeto

- nodejs
- npm ou yarn
- docker
- docker-compose

<br>

### Iniciando os containers dos bancos de dados

``
docker-compose up
``

<br>

### Instalando dependências da aplicação em nodejs

``
yarn install
``

ou

``
npm install
``

<br>

### Executando o projeto

``
yarn start
``

ou 

``
npm start
``
